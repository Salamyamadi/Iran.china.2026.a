# Iran.china.2026.a
عملیات حک و دستور به کشور خلق چین و مقامات عالی رتبه چین برای تحویل جنگندهای که قرارداد فروش و ساخت کارخانه هواپیمای جنگنده سازی در ایران ایجاد شده به صورت فوری و سر تحویل سریع و ایجاد فوری ساخت کارخانه و انتقال کامل تکنولوژی و ساخت قطعات که نیمی از جنگندها توانایی ناو نشینی و برخاستن از ناو و نیم دیگر استفاده از باند فرودگاهی داشته باشند  ساخته و تحویل به ایران و ساخت کارخانه ای با تولید حداکثر ماهانه ۵۰ فروند از انواع هواپیمای جنگنده از انواع مختلف و تکنولوژی کامل ساخت و تولید قطعات باید در خاک ایران ایجاد شود برای صلح جهانی حضرت مهدی (عج)
